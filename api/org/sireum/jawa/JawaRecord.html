org.sireum.jawa

JawaRecord

class JawaRecord extends ResolveLevel

This class is an amandroid representation of a pilar record. A record corresponds to a class or an interface of the source code. They are usually created by Amandroid Resolver. You can also construct it manually. Call init() method first when you want to do any further things.
